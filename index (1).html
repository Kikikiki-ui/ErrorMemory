🌐 About the Site

Большинство сайтов для IELTS Writing работают одинаково: ты пишешь эссе, нажимаешь "проверить" — и получаешь цифру без объяснений. ErrorMemory разворачивает этот процесс: сначала статьи и готовые грамматические конструкции по конкретной теме, потом — практика письма с разбором ошибок.

Что внутри: диагностика слабых мест, зона подготовки по темам, трекер ошибок, мок-тест с таймером, разбор грамматики по эссе

Стек: HTML5, CSS3, JavaScript
